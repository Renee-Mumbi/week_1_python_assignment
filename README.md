**JSON Data Extraction and API Integration**
**Overview**

This project demonstrates how to use Python to extract JSON data from a raw GitHub
 link and from the DummyJSON API and Mockaroo, convert the data into Pandas DataFrames, and save the results as CSV files.

**Tasks Completed**
Created a synthetic dataset using Mockaroo
Exported the dataset as a JSON file
Uploaded the JSON file to a public GitHub repository
Retrieved the data using the Requests library
Converted the JSON data into structured DataFrames
Saved the DataFrames as CSV files
Extracted additional data from DummyJSON API endpoints

**Technologies Used**      
Python
Pandas
Requests
GitHub
Mockaroo
DummyJSON API

**Output Files**
products_list.csv
view_cart.csv
staff_data.csv

**Conclusion**

This project demonstrates a complete workflow for collecting JSON data from online sources and 
converting it into structured CSV files for data analysis.
